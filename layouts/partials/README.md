# layouts/partials

PaperMod 局部模板覆盖层，用于注入字体、数学公式、Mermaid 渲染、评论与媒体增强逻辑。

| 文件 | 用途 |
| --- | --- |
| `comments.html` | 评论系统模板。 |
| `extend_footer.html` | 页面底部脚本增强，包括图片缩放和 Mermaid 放大查看。 |
| `extend_head.html` | Head 资源注入，包括字体、MathJax、Mermaid/Beautiful Mermaid。 |
| `mathjax.html` | 数学公式相关模板。 |
| `templates/` | SEO、社交卡片等模板覆盖。 |
