papers.cool/arxiv
1. EvoArena: Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments
看点：论文把智能体放进持续变化的环境里评测，并提出用“记忆补丁历史”去追踪环境演化，直击 agent 落地中的真实难点。
为什么值得看：相比静态 benchmark，这个方向更接近生产环境，对做 AI agent、长期任务和记忆系统的人很有启发｜原文：https://papers.cool/arxiv/2606.13681
2. HyperTool: Beyond Step-Wise Tool Calls for Tool-Augmented Agents
看点：核心思想是把多步工具调用折叠成更高粒度的一次执行单元，减少模型在低层数据流上的上下文浪费。
为什么值得看：这是提升 agent 工具使用效率和成功率的非常实用路线，对 MCP、工作流 agent 和多工具协同尤其关键｜原文：https://papers.cool/arxiv/2606.13663
