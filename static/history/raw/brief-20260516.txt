papers.cool/arxiv 每日 Top 5

今日论文榜单延续了 Agent 检索、模型记忆、语音智能体评测和长对话一致性等热门方向。相比纯能力堆叠，研究重心明显更偏向“让系统在真实任务里更稳、更准、更可验证”。

1. Is Grep All You Need? How Agent Harnesses Reshape Agentic Search
来源：Computation and Language｜评分：Kimi 1★
看点：论文系统比较 grep 与向量检索在多种 Agent harness 和工具调用形态下的表现，发现传统 grep 往往并不落下风。
推荐理由：这对做 Agent/RAG 的团队非常重要，它提醒大家别被“向量检索默认更先进”的直觉绑架。
关键词：grep、Agent Search、检索
原文：https://papers.cool/arxiv/2605.15184

2. Text Knows What, Tables Know When: Clinical Timeline Reconstruction via Retrieval-Augmented Multimodal Alignment
来源：Computation and Language｜评分：Top 2
看点：论文把临床文本与结构化 EHR 表格对齐，用多模态检索增强方式重建更精确的患者时间线。
推荐理由：医疗 AI 最怕时间信息失真，这项工作展示了多模态对齐如何真正改善高价值场景中的可用性。
关键词：临床时间线、多模态对齐、医疗 AI
原文：https://papers.cool/arxiv/2605.15168

3. MeMo: Memory as a Model
来源：Computation and Language｜评分：Kimi 7★
看点：MeMo 通过独立的记忆模型吸收新知识，而不直接修改主 LLM 参数，兼顾可插拔、抗遗忘与检索鲁棒性。
推荐理由：这正击中了企业知识更新的痛点，是“怎么让模型持续学新东西”这个问题上的强信号论文。
关键词：模型记忆、知识更新、检索增强
原文：https://papers.cool/arxiv/2605.15156

4. From Text to Voice: A Reproducible and Verifiable Framework for Evaluating Tool Calling LLM Agents
来源：Computation and Language｜评分：Top 4
看点：作者把文本工具调用基准转换成可验证的语音工具调用评测，用来衡量语音 Agent 在真实语音输入下的表现差距。
推荐理由：语音 Agent 正快速进入产品层，这篇论文补上了“如何严谨评测”这一块关键基础设施。
关键词：语音 Agent、工具调用、评测框架
原文：https://papers.cool/arxiv/2605.15104

5. Improving Multi-turn Dialogue Consistency with Self-Recall Thinking
来源：Computation and Language｜评分：Kimi 1★
看点：SRT 让模型在长对话中主动回忆关键历史轮次，从而提升一致性、降低延迟并减少无关上下文干扰。
推荐理由：长对话稳定性是客服、助手和陪伴场景的核心短板，这类方法非常贴近实际部署需求。
