papers.cool/arxiv 每日 Top 5

今天的 papers.cool/arxiv 榜单几乎被 LLM agent、记忆机制、对话一致性与多模态评测占据，研究热点非常集中在“让模型更会找、更会记、更会调用工具”。从题目和 Kimi 星标看，agent 检索框架与外挂记忆模块最受关注。

1. Is Grep All You Need? How Agent Harnesses Reshape Agentic Search
来源：计算与语言｜评分：Kimi 5★
看点：论文系统比较 grep 与向量检索在 agent 搜索流程中的表现，并指出检索策略与 agent harness 设计深度耦合。
推荐理由：这是当前 agent 检索栈里非常“接地气”的问题，直接关系到工具调用与长上下文任务的真实效果。
关键词：agent 搜索、grep 检索、tool calling
原文：https://papers.cool/arxiv/2605.15184

2. Text Knows What, Tables Know When: Clinical Timeline Reconstruction via Retrieval-Augmented Multimodal Alignment
来源：计算与语言｜评分：Kimi 2★
看点：论文把临床文本与结构化 EHR 表格对齐，用检索增强的多模态方法重建更精确的病程时间线。
推荐理由：这是医疗 AI 中很典型的“文本语义丰富但时间不准、表格时间准但信息不全”的融合范式，方法论价值突出。
关键词：临床时间线、多模态对齐、检索增强
原文：https://papers.cool/arxiv/2605.15168

3. MeMo: Memory as a Model
来源：计算与语言｜评分：Kimi 9★
看点：MeMo 试图把新知识编码进独立记忆模型，而不直接改动主 LLM 参数，从而降低遗忘并增强即插即用性。
推荐理由：外挂记忆是大模型实用化的重要方向，这篇工作对长期知识更新与闭源模型扩展都很有启发。
关键词：记忆模型、知识更新、检索鲁棒性
原文：https://papers.cool/arxiv/2605.15156

4. From Text to Voice: A Reproducible and Verifiable Framework for Evaluating Tool Calling LLM Agents
来源：计算与语言｜评分：Top 4
看点：论文把文本工具调用基准转换成语音场景评测，研究语音输入下工具调用 agent 的性能变化与误差来源。
推荐理由：随着语音 agent 升温，如何可验证地评估其 tool calling 能力会越来越关键，这篇工作补上了方法缺口。
关键词：语音代理、工具调用评测、多模态基准
原文：https://papers.cool/arxiv/2605.15104

5. Improving Multi-turn Dialogue Consistency with Self-Recall Thinking
来源：计算与语言｜评分：Kimi 1★
看点：论文提出 Self-Recall Thinking，让模型在多轮对话中主动回忆关键历史轮次，以改善长程依赖和一致性问题。
推荐理由：多轮对话记忆始终是产品级助手的核心瓶颈，这类内生回忆机制很值得持续关注。
关键词：多轮对话、自回忆、上下文一致性
原文：https://papers.cool/arxiv/2605.15102
