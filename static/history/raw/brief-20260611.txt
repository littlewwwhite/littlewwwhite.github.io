papers.cool/arxiv
1. PhantomBench：系统测语言模型对“不存在概念”的幻觉
看点：作者构造 6 万多个不存在的术语和实体，发现主流模型在“该不知道时”仍频繁硬答，幻觉率很高。
为什么值得看：这是评估模型边界感和拒答能力的强基准，对高风险场景尤其重要｜原文：https://papers.cool/arxiv/2606.11105
2. VISTA：面向 Agent 评测的交互式用户模拟工具包
看点：同时支持 UI 和 API 层面的用户模拟，还补上了真实度、覆盖度、有效性等评估指标。
为什么值得看：如果你在做 agent 产品，这类动态评测框架比静态 benchmark 更接近真实失败模式｜原文：https://papers.cool/arxiv/2606.11079
