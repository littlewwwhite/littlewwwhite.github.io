papers.cool/arxiv
1. AgentSpec：用可控组合拆解具身 Agent 脚手架
看点：把 perception、memory、reasoning、reflection、action 等模块标准化重组，研究 agent 性能到底来自单模块还是组合关系。
为什么值得看：对做 agent 系统的人很有价值，重点不是再堆模块，而是理解“什么搭配在什么环境下有效”｜原文：https://papers.cool/arxiv/2606.14674
2. AdaSR：自适应流式推理
看点：让模型在输入流到来时边读边想，并动态决定何时深想、分配多少计算，目标是兼顾准确率、延迟和算力效率。
为什么值得看：这比常规“更强推理”论文更贴近视频、语音、实时交互等真实场景，工程含义比较直接｜原文：https://papers.cool/arxiv/2606.14694
3. CORA：修补多模态推理里“思考过程”和“最终答案”不一致的问题
看点：给多模态 RLVR 加入一致性奖励，降低看起来想得很多、但答案和推理对不上的情况。
为什么值得看：如果多模态 agent 要进生产，faithfulness 是绕不过去的，这篇切的问题很实在｜原文：https://papers.cool/arxiv/2606.14691
