GitHub Trending
1. NousResearch/hermes-agent
看点：通用型 agent 框架，今天涨幅很猛，覆盖工具调用、记忆、子代理和多平台接入。
为什么值得看：这类“能落地的 agent 基建”比单点 demo 更有长期价值｜原文：https://github.com/NousResearch/hermes-agent
2. lfnovo/open-notebook
看点：开源版 NotebookLM，主打更灵活的知识导入、检索和对话体验。
为什么值得看：离真实生产需求更近，适合想自建私有知识助手或文档问答系统的人｜原文：https://github.com/lfnovo/open-notebook

papers.cool/arxiv
1. You Only Index Once: Cross-Layer Sparse Attention with Shared Routing
看点：把跨层共享路由用于稀疏注意力，论文声称在 128K 上可拿到 7.6x 解码加速和 17.1x 总吞吐提升。
为什么值得看：这是少数同时碰“长上下文质量”和“推理成本”两个核心瓶颈的工作｜原文：https://papers.cool/arxiv/2606.06467
2. Self-Augmenting Retrieval for Diffusion Language Models
看点：利用扩散语言模型去噪过程中的低置信 token 提前做检索，形成训练免费、可插拔的动态 RAG。
为什么值得看：不仅新，而且实用指标漂亮；如果扩散式 LLM 继续升温，这会是很值得盯的一条技术线｜原文：https://papers.cool/arxiv/2606.06474
