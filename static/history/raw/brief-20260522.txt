papers.cool/arxiv 每日 Top 5

今天的论文榜单以 NLP、LLM agent 和记忆机制为主，偏“语言理解 + 代理能力 + 软件工程自动化”三线并进。唯一带 Kimi 星标的是一篇关于自适应记忆的 agent 论文，信号很明确：让模型知道“何时记、记什么”正在成为新热点。

1. Leveraging LLMs for Grammar Adaptation: A Study on Metamodel-Grammar Co-Evolution

看点：论文研究如何用 LLM 自动适配元模型演化后的语法，显著优于部分规则式方法，尤其适合复杂 DSL 演化场景。
推荐理由：它把 LLM 用到软件工程自动化的细分难题里，展示了模型在“理解并修改形式化结构”上的实际潜力。
关键词：语法适配、元模型、软件工程
原文：https://papers.cool/arxiv/2605.21465

2. Mem-π: Adaptive Memory through Learning When and What to Generate

推荐理由：这正击中 LLM agent 的核心瓶颈，尤其对网页导航、终端工具使用等复杂任务很有参考价值。
