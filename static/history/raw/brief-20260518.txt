papers.cool/arxiv 每日 Top 5

今天的论文榜单集中在 Agent 检索、医学多模态时间线、LLM 外挂记忆、语音工具调用与多轮对话一致性，整体上很偏“让模型在真实任务里更可靠”。从研究走向看，大家已经不满足于单纯提升基准分数，而是在系统性补齐可用性与可验证性。

1. Is Grep All You Need? How Agent Harnesses Reshape Agentic Search
来源：Computation and Language｜评分：Kimi 6★
看点：论文比较 grep 与向量检索在 Agent 搜索中的表现，指出具体 Agent harness 与工具调用形态会显著影响结果。
推荐理由：它直接切中当下 Agent 系统最实际的问题之一：检索策略和工具接口到底怎样影响最终任务完成率。
关键词：Agent检索、Grep、工具调用
原文：https://papers.cool/arxiv/2605.15184

2. Text Knows What, Tables Know When: Clinical Timeline Reconstruction via Retrieval-Augmented Multimodal Alignment
来源：Computation and Language / Artificial Intelligence / Machine Learning｜评分：Kimi 4★
看点：研究把临床文本与结构化 EHR 表格对齐，用检索增强的多模态方法提升患者时间线重建精度。
推荐理由：这类把大模型真正嵌入医疗时序理解的工作，往往比通用 benchmark 提升更接近真实影响。
关键词：临床时间线、多模态对齐、医疗AI
原文：https://papers.cool/arxiv/2605.15168

3. MeMo: Memory as a Model
来源：Computation and Language / Artificial Intelligence / Machine Learning｜评分：Kimi 10★
看点：MeMo 提出把新增知识编码进独立记忆模型，而不直接改动主 LLM 参数，以降低遗忘并提升可插拔性。
推荐理由：对于希望让模型持续吸收新知识、又不想频繁重训主模型的团队，这是很有启发性的架构方向。
关键词：模型记忆、知识更新、可插拔LLM
原文：https://papers.cool/arxiv/2605.15156

4. From Text to Voice: A Reproducible and Verifiable Framework for Evaluating Tool Calling LLM Agents
来源：Computation and Language｜评分：Top 4
看点：论文把文本工具调用基准转成可控语音版本，用来评估语音 Agent 在真实噪声场景下的工具使用能力。
推荐理由：随着语音助手和实时 Agent 升温，这类“可复现、可验证”的评测框架会越来越重要。
关键词：语音Agent、工具调用、评测框架
原文：https://papers.cool/arxiv/2605.15104

5. Improving Multi-turn Dialogue Consistency with Self-Recall Thinking
来源：Computation and Language / Artificial Intelligence｜评分：Kimi 3★
看点：论文提出 Self-Recall Thinking，让模型在多轮对话中主动召回关键历史上下文，以提升一致性并降低延迟。
推荐理由：它回应了长对话系统最常见的痛点之一：信息越长越散、上下文越多越慢。
关键词：多轮对话、自回忆推理、一致性
原文：https://papers.cool/arxiv/2605.15102
