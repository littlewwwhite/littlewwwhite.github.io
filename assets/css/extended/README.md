# assets/css/extended

PaperMod 的项目级样式覆盖层，负责字体、主题变量、内容排版、代码块、表格、数学公式和 Mermaid 图表样式。

| 文件 | 用途 |
| --- | --- |
| `blank.css` | PaperMod 预留空扩展文件。 |
| `chroma-styles-overrides.css` | Chroma 语法高亮 token 配色覆盖。 |
| `custom.css` | 文章内容排版、表格、代码块、内联高亮和 Mermaid 样式。 |
| `footer-overrides.css` | 页脚样式覆盖。 |
| `math.css` | 数学公式显示样式。 |
| `scroll-bar-overrides.css` | 滚动条样式覆盖。 |
| `theme-vars-overrides.css` | PaperMod 主题变量、字体和明暗色语义变量。 |
